## 1. Development & Terminal

These are **the REAL heavy hitters** for coding, debugging, and system interaction - my daily driver tools.

- **code:** Visual Studio Code - My go-to IDE for everything
- **gcc / gcc-c++:** The GNU C/C++ compiler - For when I need to compile from scratch
- **git:** Version control - My code time machine
- **starship:** Your customized shell prompt - Makes terminal beautiful
- **btop / htop / atop:** System resource monitors - Keep an eye on system health
- **nvtop:** NVIDIA GPU monitor (if NVIDIA drivers are installed) - GPU performance tracking
- **gdb / strace / valgrind:** Debugging and profiling tools - When things go wrong
- **make / autoconf / automake:** Build automation - The classic build tools
- **ccache:** Compiler cache for faster rebuilds - Speed up my compilation time

## 2. Media & Graphics

- **gimp:** Professional image manipulation - My Photoshop alternative
- **steam:** Gaming platform - Because work-life balance matters
- **vlc:** The "play anything" media player - Never met a format it can't handle
- **ffmpeg:** Powerful command-line video/audio converter - My media swiss army knife
- **gamemode:** Optimizes system performance for gaming - When I need those extra FPS
- **openrgb:** Control for RGB lighting on your hardware - Make my setup shine
- **nvtop:** Monitor your NVIDIA/AMD GPU usage - Keep tabs on my GPU performance

## 3. Internet & Networking

- **chromium / firefox:** Your web browsers - My portals to the internet
- **nmap:** Network scanning and security auditing - Network exploration and security
- **wget2 / curl:** Command-line file downloaders - My URL toolkit
- **transmission:** (Likely used via GNOME) BitTorrent client - For file sharing
- **bind-utils:** Network diagnostic tools like dig and nslookup - DNS troubleshooting

## 4. System & Desktop Environment

- **akmod-nvidia:** The NVIDIA proprietary drivers - Making my graphics work
- **gnome-shell / gnome-tweaks:** The core UI and its customization tool - My desktop, my rules
- **fastfetch:** Displays system info in the terminal (the "Flex" tool) - Show off my system specs
- **plymouth:** The boot animation/splash screen - Making startup beautiful
- **zsh / bash:** Your command-line shells - My command playground
- **zsh-autosuggestions / zsh-syntax-highlighting:** Logic for your Zsh shell - Smarter terminal experience
- **virt-manager / gnome-boxes:** Virtual machine management - For testing and isolation

## 5. Productivity & Utilities

- **libreoffice:** Complete office suite - When Microsoft Office isn't an option
- **baobab:** Disk usage analyzer (GNOME Disks) - Where did all my storage go?
- **rclone / rsync:** Advanced file backup and synchronization - My data protection toolkit
- **fpaste:** Fedora's pastebin for sharing logs - For when I need help
- **simple-scan:** Document scanning utility - Digitize my paper world
